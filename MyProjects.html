Cybersecurity Web Application

This project was created as part of a Cybersecurity class at "HIT" college.

The project is a web application that "sells" mobile data packages.

It includes features such as registration, login, a home page, a "Forgot My Password" feature, and a "Change My Password" feature.

The app is connected to a MySQL server for managing users and "packages," as well as an Azure server for hosting.

Additionally, it integrates with the MailJet service to send emails containing randomized passwords to users

who have forgotten their passwords.

The primary focus of the application is to prevent security vulnerabilities such as SQL injection attacks and cross-site scripting (XSS) attacks.

This was achieved using several techniques, including password hashing, enforcing advanced password criteria, encrypting SQL queries, and utilizing cookies for user recognition.
